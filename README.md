# 幸运大转盘

## 操作说明

### 基础操作

1. 点击左上角文字打开配置页面
2. 添加待选数据(如数据未分配 100%, 则剩余的部分会以再来一次的形式出现)
3. 选择转盘旋转时间的范围
4. 选择转盘的运动曲线
5. 点击更新转盘

### 数据同步(上传)

1. 数据表格不为空
2. 点击同步配置
3. 输入名称新建配置项，或选择已有配置更改其数据

### 数据同步(拉取)

1. 清空数据表格
2. 点击同步配置
3. 选择一份配置项
